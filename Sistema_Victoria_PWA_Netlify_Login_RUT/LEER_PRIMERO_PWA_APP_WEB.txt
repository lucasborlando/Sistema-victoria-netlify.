SISTEMA VICTORIA - VERSIÓN INFORMATIVA COMO APP WEB (PWA)

1) Para Netlify, sube la carpeta:
   Sistema_Victoria_NETLIFY_Informativo

2) En Netlify configura:
   Publish directory: .

3) La versión Netlify es solo lectura:
   - Permite ver inventario.
   - Permite ver historial.
   - Permite descargar Excel.
   - No permite modificar inventario ni usuarios.

4) Instalación en celular Android:
   - Abre la URL de Netlify en Chrome.
   - Presiona el botón INSTALAR APP si aparece, o menú ⋮ > Agregar a pantalla principal.

5) Instalación en iPhone:
   - Abre la URL en Safari.
   - Presiona Compartir > Agregar a pantalla de inicio.

6) Actualización de datos:
   - Trabajas en la versión local desde VS Code.
   - Ejecutas la sincronización/exportación hacia la carpeta Netlify/GitHub.
   - Netlify republica la versión informativa automáticamente.

IMPORTANTE:
La app web guarda archivos básicos para abrir como aplicación y mejorar carga en celular, pero los datos JSON se consultan con prioridad desde internet para evitar ver información antigua cuando haya cambios nuevos.
